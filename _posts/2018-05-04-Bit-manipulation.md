有时候，位运算相当于一种小技巧，用得合适，程度会非常简单漂亮~

#### 基本位运算
| 运算|运算符 | 含义 | 
| ------------- |:-------------:| :-------------:| 
| 与 AND | & | 1&1=1| 
| 或 OR | \| | 0\|0=0 |  
| 异或 XOR | ^ | 1^1=0, 0^0=0 | 
| 非 NOT | ~ | 一元运算 | 
| 左移 | << | 二元运算,乘2 | 
| 右移 | >> | 二元运算,除以2 | 

#### 使用技巧总结
参考博客：https://blog.csdn.net/zmazon/article/details/8262185
1. 求int类型最大值
```c++
int max_int= ((1 << 31) - 1);
// 或者 ~(1 << 31);//2147483647
```
2. 求int类型最小值
```c++
int max_int= (1 << 31 );//-2147483648
```
3. 乘以2的m次方
```c++
n = (n<<m);
```
4. 除以2的m次方
```c++
n = (n>>m);
```
5. 判断一个数的奇偶性
```c++
(n&1)==1?1:0;
```
6. 不使用临时变量交换两个数
```c++
a ^= b;  
b ^= a;  
a ^= b; 
```
7. 判断符号是否相同
```c++
(x ^ y) >= 0?1:0;
```
8. 判断一个数是不是2的幂数
```c++
n > 0 ? (n & (n - 1)) == 0 : false; 
```
9. 求平均值
```c++
// 直接求(x + y) / 2，x＋y可能超过int的最大范围 
int x、y;
int mid = (x&y)+((x^y)>>1); 
```
